## **Elementor Integration**

Our theme is fully compatible with the **Elementor Page Builder**, allowing you to create stunning, responsive layouts visually — without writing a single line of code. With Elementor, you can design pages, posts, and custom templates quickly and intuitively.

For detailed guidance on using Elementor, please refer to the official [**Elementor Documentation**](https://elementor.com/help/).

### **Elementor Settings**

Before you start creating pages with Elementor, make sure that the relevant post types are enabled for Elementor editing.

To configure this:

1. Navigate to **Dashboard → Elementor → Settings**.  
2. Under the **General** tab, check the boxes for the post types you want to edit with Elementor (e.g., **Posts**, **Pages**, **Service**, **Project**, **Team**).  
3. Click **Save Changes** to apply your settings.

![Elementor Settings](/assets/domio/images/elementor-settings/2.png)

This ensures that Elementor can be used to edit and customize all selected post types seamlessly within your WordPress dashboard.
