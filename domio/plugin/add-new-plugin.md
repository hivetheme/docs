## Add New Plugin

Upload Core Plugin
<strong># Step 01:</strong>
Click on Add New plugins to install Core plugins.
<p><img src="/assets/domio/images/install-plugins/core-1.png"></p>
<br>
<br>
<strong># Step 02:</strong>
Click Choose File and select domio-core.zip file.
<p><img src="/assets/domio/images/install-plugins/core-2.png"></p>
<br>
<br>
<strong># Step 03:</strong>
Click Install Now Button
<p><img src="/assets/domio/images/install-plugins/core-3.png"></p>
<br>
<br>
<strong># Step 04:</strong>
Click Active plugins
<p><img src="/assets/domio/images/install-plugins/core-4.png"></p>
